Google Calendar Setup

To use Google Calendar sync, you need to:

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Google Calendar API
  4. Create OAuth 2.0 credentials (Web application)
  5. Add your Client ID to manifest.json in the oauth2.client_id field
  6. Add chrome-extension://YOUR_EXTENSION_ID to authorized redirect URIs

Note: The extension will prompt you to authenticate when you first sync to Google Calendar.
